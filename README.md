\# Lung Cancer Classification using Custom CNN



This repository contains a Deep Learning project for classifying lung cancer from medical images using a custom Convolutional Network (CNN) architecture.



\## 🚀 Key Features

\- \*\*Custom CNN Architecture:\*\* Built from scratch to efficiently extract spatial features from lung scan images.

\- \*\*Overfitting Control:\*\* Implemented \*\*Dropout\*\* regularization layers to ensure the model generalizes well.

\- \*\*High Performance:\*\* Achieved an outstanding \*\*95% Validation Accuracy\*\*.



\## 📁 Repository Structure

\* `main.ipynb` - Jupyter Notebook containing data preprocessing, model architecture, training, and evaluation.

\* `app.py` - Application script for model deployment/inference.

\* `model\_confusion\_matrix.png` \& `test\_confusion\_matrix.png` - Visual evaluation metrics.

\* `.gitignore` - Configurations to avoid tracking unnecessary files.
